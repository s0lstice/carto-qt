bool initPointsTable ( )

fonction creant la table points dans la base de donnée

Renvoie:
vrai|fau : bool : indique la reussite ou l'echque de l'initialisation
Note:
les champs de la table points sont :
-point_id : INTEGER PRIMARY KEY AUTOINCREMENT : clef primaire de latable
-name : VARCHAR(64) : nom d'un point, 64 caratere maximum
-categorie_id : INTEGER : clef etrangere sur la table categorie
-latitude : FLOAT : latitude du point
-longitude : FLOAT : longitude du point
-description : VARCHAR(1024) : description du point, 1024 caractere maximum
info : les champs name, latitude, longitude, categorie_id forme un group unique.
