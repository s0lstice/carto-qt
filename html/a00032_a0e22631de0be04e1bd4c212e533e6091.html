bool initJoursTable ( )

fonction : creant la table heures dans la base de donnée

Renvoie:
vrai|fau : bool : indique la reussite ou l'echque de l'initialisation
Note:
les champs de la table points sont :
-jour_id : INTEGER PRIMARY KEY AUTOINCREMENT : clef primaire de latable
-jour : VARCHAR(64) : nom du jour, 64 caratere maximum
info : le champ jour est unique.
Tout comme dans sql au niveau de la gestion des jours, ceci vont de dimanche a samedi avec des identifiant de 1 a 7.
Les jours sont contenuent dans un QStringList et permetent leurs traduction. a ce niveau, il et donc paussible de donner une langue a la base de donnee.

Définition à la ligne 21 du fichier data_jours.cpp.

                     {
    QSqlQuery query(database::dataCreate()->dataConnect());

    QStringList jours;
    jours << QObject::tr("dimanche")
            << QObject::tr("lundi")
            << QObject::tr("mardi")
            << QObject::tr("mercredi")
            << QObject::tr("jeudi")
            << QObject::tr("vendredi")
            << QObject::tr("samedi");

    query.exec("create table jours ("
                  "jour_id INTEGER PRIMARY KEY AUTOINCREMENT, "
                  "jour VARCHAR(64),"
                  "unique (jour))");


    foreach(QString jour, jours){
        query.exec("insert into jours(jour) values('" + jour + "')");
    }
    return true;
}
